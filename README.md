**MyProject** is an Entity-Component Engine that I decided to make as my first personal project. It will be done using my knowledge from my career, and will try to include as many things as possible. The goal is to make it an almost-generic engine that I can use for most kind of games.

## **Objectives**

As mentioned, my plan is to make an almost-generic engine to be used on the creation of most kind of games. For that, after building the Engine my plan is to try to create games of different genres using my Engine. From small to medium projects.
